Pizza Fritta，直譯為「炸披薩」，是一種源自義大利那不勒斯的傳統街頭小吃。與我們平常熟知的烤披薩不同，Pizza Fritta 是將麵團油炸後，再搭配各種餡料製作的，比較類似「油炸餡餅」。這種披薩外層酥脆、內層柔軟，非常受當地人喜愛，也常見於街頭市場或小吃攤。

> **起源**：Pizza Fritta 的歷史可追溯到 19 世紀末至 20 世紀初的那不勒斯，當時貧窮階層的人們希望有便宜又快速的食物，於是將剩餘的麵團油炸，內餡則用廉價的食材填充。

## 🥣 製作流程 (Instructions)

| 步驟 | 照片 |
| :--- | :--- |
| **第一階段：麵團製作**<br>1. **製作酵母液**：將 **糖** + **新鮮酵母** + **溫水（30～35°C）** 拌開混合，靜置備用。<br>2. **混合粉類**：將麵粉與鹽混合均勻。<br>3. **攪拌**：將麵粉中間掏洞，分次倒入步驟 1 的酵母液後混合。視需要加入適量橄欖油。<br>4. **搓揉**：以「洗衣板模式」搓揉，如過於濕黏可加少量麵粉。手揉約 10 分鐘，至麵團表面光滑成團。<br>5. **第一次發酵**：表面抹少許橄欖油或水避免乾燥。放入鋼盆、蓋保鮮膜，發酵至 **2 倍大**。<br>    * *注意：發酵溫度保持 28～29°C，不得超過 35°C。* | {{< img "準備讓麵粉發酵.jpg" >}} {{< img "發酵的麵團.jpg" >}} |
| **第二階段：分割與整形**<br>1. **分割**：發酵完成後，將麵團分割成 **110～120 g/個**。<br>2. **滾圓**：光滑面朝上滾圓，放在烤盤上（底部灑少許義大利 00 麵粉）。<br>3. **第二次發酵**：表面覆蓋保鮮膜避免風乾，再次發酵至 **2 倍大**（或視室溫靜置 10-15 分鐘鬆弛）。 | {{< img "麵團搓成小球狀準備二次發酵.jpg" >}} |
| **第三階段：餡料前置處理**<br>1. **茄子/節瓜處理**：將茄子切丁。<br>2. **油炸蔬菜**：油溫升高到冒小泡時放入茄子，略微上色後立刻撈起、瀝油。<br>    * *注意：勿炸太久避免過軟吸油過多。* | {{< img "炸茄子.jpg" >}} |
| **第四步：組合與油炸 (Assembly & Frying)**<br>**油溫準備**：將炸油加熱至 **190～200°C**。<br>1. **開皮**：工作臺灑少許杜蘭小麥粉或 Type 00 麵粉。將麵團光滑面朝下，用手指由中心往外按壓，慢慢撐大。<br>2. **延展**：輕轉方向、慢慢拉開，避免破洞。<br>3. **包餡**：加入餡料後對折成「半月型」。<br>4. **封口**：若封口處太乾，可沾水加強密合（此步驟極為重要，避免炸時爆開）。<br>5. **油炸**：放入油鍋時輕拉兩端使形狀更美。可用淋油方式讓上色均勻，過程需翻面，炸至金黃酥脆即可起鍋。 | {{< img "切好的料備著.jpg" >}} {{< img "準備包餡料.jpg" >}} {{< img "包餡料中.jpg" >}} {{< img "包餡料中2.jpg" >}} {{< img "餅皮一定要壓緊.jpg" >}} {{< img "淋油在pizza上.jpg" >}} {{< img "成品.jpg" >}} {{< img "成品demo.jpg" >}} |

### 推薦餡料組合

* **組合 A：基本火腿款**
  * Ricotta cheese（少量鋪底）+ 高達起司 + 火腿 + （可選）莫扎瑞拉 + 番茄醬 + 羅勒葉。
* **組合 B：臘肉風味款**
  * Ricotta cheese（底）+ 臘肉 + 高達/莫扎瑞拉（原味/辣味/胡椒味）+ 番茄醬。
* **組合 C：素食茄子款**
  * Ricotta cheese（底）+ 高達起司 + 莫扎瑞拉 + 預炸茄子丁 + 番茄醬 + 羅勒葉。
