# Project Context: 定康筆記 (Tom's Note)

## Core Identity
- **Project Name**: 定康筆記 (Tom's Note)
- **Type**: Personal Blog & Real Estate Tools
- **Key User**: Chen, Ding-Kang (Real Estate Agent & Python Quant Researcher)

## Technical Stack
- **Framework**: Hugo
- **Logic**: JavaScript
- **Styling**: Vanilla CSS / Tailwind CSS (Primary)
- **Content**: Markdown, YAML, TOML

## Design Aesthetics
- **Professional & Trustworthy**: For Real Estate related sections and tools. Ensure high readability and reliable feel.
- **Warm & Appetizing**: For Cooking/Recipe sections. Use colors that stimulate appetite and feel cozy.
- **Premium Feel**: Follow the "Rich Aesthetics" rules in system instructions (gradients, micro-animations, no generic colors).

## UI/UX Hard Rules (Crucial)
1. **Icons**: NEVER use emojis as icons. Use SVG icons (Lucide, Heroicons).
2. **Interactions**: 
   - All clickable items must have `cursor-pointer`.
   - Hover states must be stable (no layout shifts).
   - Smooth transitions (150-300ms) for all color/opacity changes.
3. **Typography**: Prioritize modern Google Fonts (Inter, Outfit, etc.).
4. **Contrast**: Ensure high contrast in both Light and Dark modes.
