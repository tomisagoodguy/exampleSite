## Coding Standards

- **Naming**:
  - Components: PascalCase
  - Functions/Variables: camelCase
  - CSS Classes: kebab-case
- **Modularity**: Prioritize reusable partials in `layouts/partials/`.
- **Workflow**:
  - Use `.agent/workflows/` for SOPs.
  - Follow TDD/AAA for any script testing.
  - No `any` in TypeScript.
